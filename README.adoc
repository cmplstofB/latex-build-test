== しくみ

* 手許でLaTeX原稿を編集する
例えば，
[source,console]
----
$ editor main.tex
----

* git標識を賦与する。
例えば，
[source,console]
----
$ git tag v42.0.1
----

* GitHubに押し込む。
例えば，
[source,console]
----
$ git push origin v42.0.1
----

ここでGitHub Actionsが作動。
「pushかつ “v” から始まるtagが賦与されていれば
指定した一連の動作（具体的にはLaTeX組版と
Releaseへの上げ込み）を行う」
という設定にしているので，
如述の操作に感応する。

Dockerが構築されて原稿が処理され，
しばらく待つと
Releaseが一つ増えてそこに
組版結果のPDF文書が上がる。
